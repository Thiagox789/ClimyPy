Acerca de ClimyPy

ClimyPy es un sistema de monitoreo ambiental desarrollado con tecnología ESP32 y sensor DHT11. Proporciona mediciones precisas de temperatura y humedad en tiempo real a través de una interfaz web moderna.

Características principales:

  • Monitoreo inalámbrico via WiFi
  • Interfaz web responsive y moderna
  • Gráficos históricos interactivos
  • Base de datos SQLite para almacenamiento
  • API REST para integración

Desarrollado con amor por el equipo de ClimyPy para proporcionar una solución simple y efectiva para el monitoreo ambiental.
